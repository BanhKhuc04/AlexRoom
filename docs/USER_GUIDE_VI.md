# Hướng dẫn sử dụng ALEX NEXUS OS

- Presence: chạm Core hoặc nhấn Space để mở kênh lệnh; Escape/đóng để trở về idle.
- Chuyển giao diện bằng nút ALEX/PRESENCE hoặc `Ctrl+Space`; trạng thái không reload.
- Command Center có 12 workspace. Nhãn `UNKNOWN`, `PENDING`, `NOT VERIFIED` là trạng thái trung thực, không phải lỗi trang trí.
- Cài đặt: chọn quality, reduced motion, sound mode và gain. Sound chỉ mở sau thao tác người dùng do chính sách autoplay.
- Điều khiển relay cần API key và reported-state ACK. Không dùng với tải nguy hiểm chưa kiểm tra.
- Microphone cần browser permission; nếu không cấp quyền, waveform semantic vẫn hoạt động nhưng không đại diện âm thanh thật.
