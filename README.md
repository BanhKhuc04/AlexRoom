# ALEXROOM CODEX CONTEXT PACK

This pack is intended to be copied into the root of the `AlexRoom` repository.

Files:
- `AGENTS.md` — durable project instructions Codex should read before tasks.
- `docs/PRODUCT_SPEC.md` — product, UI, animation, sound, architecture, and safety requirements.
- `docs/IMPLEMENTATION_PLAN.md` — phased delivery plan and quality gates.
- `CODEX_FIRST_PROMPT.txt` — paste into the first Codex thread.
- `CODEX_PHASE_2_PROMPT.txt` — paste only after Phase 0/1 is complete.

Recommended flow:
1. Back up or commit the current repository.
2. Copy this pack into the repository root.
3. Open the AlexRoom project in Codex.
4. Paste `CODEX_FIRST_PROMPT.txt`.
5. Review the audit and real screenshots.
6. Continue with the Phase 2 prompt.

Do not ask Codex to implement every phase in one thread. Focused phases usually produce better reviewable changes.
